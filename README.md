# songs-teller

`songs-teller` is an audio streaming application designed to enrich the listening experience by providing deep insights, narratives, and stories behind songs and the artists who performed them.

## Features

- **Narrative Audio Streams**: Listen to curated stories, historical context, and technical insights about your favorite tracks.
- **Artist Biographies**: Discover the life and career milestones of the artists behind the music.
- **Music News & Updates**: Stay informed with the latest news and trending stories in the music industry.
- **Seamless Integration**: High-quality audio playback that blends music with spoken-word narratives.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/songs-teller.git
   ```

2. Navigate to the project directory:

   ```bash
   cd songs-teller
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

### Running the App

Start the development server:

```bash
npm start
```
